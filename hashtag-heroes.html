About

Hashtag Heroes is a 2D infinite runner game with Twitter Search integration, built using Unity and C#. You play as a robot flying through the sky, who has to dodge other evil robots flying towards it!

The player enters a Twitter hashtag at the start of the game, and each enemy robot represents a tweet from that hashtag in reverse chronological order. When you crash into a robot, it shows you that robot's tweet. The game was made in 5 days for Young Rewired State Festival of Code 2015.

My role

I was the sole developer on this project, creating all the art assets and implementing all the code used in the game.
